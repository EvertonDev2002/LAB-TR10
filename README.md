## Primeiros passos
Primeiramente é necessário ter o `gcc` em sua máquina.

## Download
```
$ git clone https://github.com/EvertonDev2002/LAB-TR10.git 
```

## Compilação
Linux
```
$ gcc TR3_537858.c -o run
```
Windows
```
$ gcc TR3_537858.c -o run.exe
```

## Execução
Linux:
```
$ ./run
```

Windows:
```
$ ./run.exe
```
